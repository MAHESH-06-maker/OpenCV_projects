This is a beginner-level OpenCV project that I built as I started learning computer vision. It is a real-time drawing application that allows users to draw on the screen using a colored object (such as a blue marker) tracked through a webcam.

## Features

* Draw using finger or colored object movement
* Eraser mode (E)
* Pause/Resume drawing (W)
* Clear canvas (C)
* Live webcam tracking

## Technologies Used

Python, OpenCV, NumPy

## Controls

* Q → Quit
* C → Clear canvas
* E → Eraser mode
* D → Draw mode
* W → Pause/Resume

## Description

This project demonstrates basic computer vision concepts such as color detection, contour tracking, and real-time drawing. It is a simple starting point for exploring OpenCV and building interactive vision-based applications.
